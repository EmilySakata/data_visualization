Comparing the 2017 San Francisco marathon race results.


Strava project is a python data analysis performed to understand key questions we had below from the San Francisco Marathon race of 2017. To answer below key questions, we used techniques such as data scraping, data cleansing, and data visualization using python programing language.

Key questions are:

  • What does the Strava user population look like compared to overall marathon runners?
  • How did Top20 Strava users perform on the race day ?
  • What can we do to race for better performance?
